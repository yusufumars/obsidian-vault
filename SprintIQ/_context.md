# SprintIQ — Sprint Capacity Planning Tool

**Status**: Active — production on Vercel  
**Repo**: github.com/yusufumars (check for exact repo name)  
**Last updated**: 2026-04-12

## What it is
- Personal sprint capacity planning tool
- Used by me as a PM learning lab for data-driven decision making and RICE scoring

## Stack
- React + Vite + Tailwind CSS v3
- Supabase (database + auth)
- React Router
- Hosted on Vercel

## Current features
- Multi-team sprint planning
- Capacity calculation
- Leave management with date pickers
- Public holidays support
- Utilisation badges
- Velocity tracking page

## Planned features
- Jira-to-SprintIQ sync via n8n
- Embedded help chatbot
- UI redesign: dark background + neon lime green
- Playwright testing
- PostHog (product analytics)
- Sentry (error tracking)
